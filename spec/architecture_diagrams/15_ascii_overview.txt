================================================================================
                              核心设计亮点
================================================================================

  ┌──────────────────────────────────────────────────────────────────────────┐
  │  ⚡ 热路径零 DB 读取                                                       │
  │     Tick 处理只读内存 InventoryStateManager                               │
  │     成交先更新内存，异步队列批量持久化                                     │
  ├──────────────────────────────────────────────────────────────────────────┤
  │  🔮 统一定价 Oracle                                                       │
  │     YES 侧计算 FV，NO 侧派生                                              │
  │     动态 Spread: base_spread × (1 + |OBI|)                               │
  ├──────────────────────────────────────────────────────────────────────────┤
  │  🔄 差分报价机制                                                          │
  │     只撤不一致订单，只补缺失档位                                           │
  │     三重保护: Lifetime + 价格偏移 + Rewards Band                          │
  ├──────────────────────────────────────────────────────────────────────────┤
  │  🛡️ 四层风控体系                                                          │
  │     报价前预检 → Watchdog 硬熔断 → REST 对账 → 硬重置                    │
  ├──────────────────────────────────────────────────────────────────────────┤
  │  🤖 智能自动路由                                                          │
  │     评分算法: ROI × rate × (10000/liquidity) × time_decay               │
  │     赛道隔离 + 事件地平线                                                │
  ├──────────────────────────────────────────────────────────────────────────┤
  │  🔧 V6.4 Ghost Order 防护                                                │
  │     每 5 分钟硬重置: CLOB cancel_all + 本地清理 + 强制对账               │
  └──────────────────────────────────────────────────────────────────────────┘

================================================================================
                               核心参数一览
================================================================================

  单市场敞口上限:     MAX_EXPOSURE_PER_MARKET     $50
  全局资金上限:       GLOBAL_MAX_BUDGET           $1000
  对账容差:           EXPOSURE_TOLERANCE          0.01 (1%)
  时间保护窗口:       RECONCILIATION_BUFFER       8 秒
  对账间隔:           RECONCILIATION_INTERVAL     60 秒
  硬重置间隔:         HARD_RESET_INTERVAL         5 分钟
  事件地平线:         EVENT_HORIZON_HOURS         24 小时
  熔断阈值:           CIRCUIT_BREAKER_FAILURES   5 次
